# homebrew-fileuni

Official Homebrew tap for FileUni CLI and GUI packages.

This repository is generated by the FileUni release workflow.
Do not edit generated formula or cask files manually.

## What You Get

- Formula: `fileuni` (CLI)
- Cask: `fileuni-gui` (GUI)
- Supported platforms:
  - `fileuni`: macOS Apple Silicon, macOS Intel, Linux x86_64 (GNU libc), Linux arm64 (GNU libc)
  - `fileuni-gui`: macOS Apple Silicon, macOS Intel

## Install

```bash
brew tap FileUni/fileuni
brew install fileuni
brew install --cask fileuni-gui
```

## Upgrade

```bash
brew update
brew upgrade fileuni
brew upgrade --cask fileuni-gui
```

## Verify

```bash
fileuni --help
open -a "FileUni UI"
```

## Uninstall

```bash
brew uninstall fileuni
brew uninstall --cask fileuni-gui
```
